# BookMyShow Frontend

This is a minimal React frontend (Vite) for the BookMyShow backend.

Prerequisites:
- Node 18+ (npm)
- Backend running at `http://localhost:8080`

Install & run:

```bash
cd frontend
npm install
npm run dev
```

The dev server runs at `http://localhost:5173` by default.

API URL can be configured with `VITE_API_URL` env var (e.g. `VITE_API_URL=http://localhost:8080`).
